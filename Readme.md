## Installation

### Prerequisites

- Node.js (version 18.0.0 or higher)
- npm (version 10.0.0 or higher)

### Steps

1. Clone the repository:
    ```bash
    git clone https://github.com/Ankit-Dwij/wallpaper-engine.git
    cd wallpaper-engine
    ```

2. Install the dependencies:
    ```bash
    npm install
    ```

3. Start the application:
    ```bash
    npm start
    ```
